# Codex Terminal

A VS Code extension to quickly launch Codex in a split terminal.

## Features

- Opens a new terminal in the right editor panel
- Automatically runs the `codex` command
- Click the sidebar icon or use keyboard shortcut

## Usage

- **Sidebar**: Click the Codex Terminal icon in the activity bar
- **Command Palette**: `Cmd+Shift+P` → "Open Codex Terminal"
